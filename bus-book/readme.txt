Wordpress Simple Bus Ticket Booking Plugin

== Description ==

A Simple Bus Ticket Booking Plugin. Day-Wise Booking.


== Installation ==

Requirements
PHP 7.2 or greater is recommended
MySQL 5.6 or greater is recommended

Upload via WordPress Admin. You can easily add a new plugin by uploading a zip archive of the plugin from your local computer.

e.g.

1. Upload `bus-book.zip` to the `/wp-content/plugins/` directory via WordPress Admin.
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Place `[bus_book_form]` in your templates

Manual Plugin Installation

Manual installation method requires downloading the WooCommerce plugin and uploading it to your web server via your favorite FTP application. The WordPress codex contains [instructions on how to do this here](https://wordpress.org/support/article/managing-plugins/#manual-plugin-installation)
